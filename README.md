# WebGL-Visual-Learning-CW
Coursework for Visual Learning written in WebGL.
Demonstrates 3D Object manipulation with the Stanford bunny for the first section of the assignment and an application of Perlin Noise applied to sphere using shaders for the "do something cool" requirement. Technical report pdf now included offering full explanation of methodology and in depth review of Perlin Noise. Final grade 100%. Script will run in browser given a WebGl compatible server.
